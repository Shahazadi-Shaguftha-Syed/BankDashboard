# Mifos Pocket Dashboard

A Proof of Concept for the Mifos Pay Pocket Feature (issue #1997),
built to demonstrate understanding of the Fineract API and Pocket UI
before contributing to the actual KMP codebase.

## Features
- Unified dashboard showing all linked accounts with aggregated balance
- Link and delink savings, loan, and share accounts
- Net worth calculator (Assets − Liabilities)
- Transaction history per account
- Multi-currency support (USD, INR, EUR, GBP, BDT)
- Budget alerts with SQLite persistence
- Portfolio breakdown chart

## Tech Stack
Python, Flask, SQLite, HTML, CSS, JavaScript
